# Source Trust Policy

## Priority order

1. Official bank page, PDF, or API
2. Official FAQ / notice / campaign page
3. Secondary comparison or content sites

## Secondary-source rules

Secondary sources may help with:

- finding hidden plan structures
- spotting merchant lists
- identifying missing percentages or exclusions

Secondary sources must not be treated as final truth for production data unless confirmed by official sources.

## Safety rule

If a fact changes recommendation output or reward percentage:

- confirm it from an official source before treating it as production truth

If not confirmed:

- keep it as review evidence
- or classify the rule as `CATALOG_ONLY`
- or record it as a schema/runtime gap
